介绍

ZSTUCA

创立于 2001 年，浙江理工大学校历史较为悠久的社团之一。伴随浙理工这所优秀的大学而成长。我们是 ZSTUCA， 一个非典型高校的非典型人类聚集体。我们爱 Coding 也爱Creating ，爱 Creating 也爱 Sleeping。我们秉承开源共享，协作与开发，创意与创造的极客风尚。

WE

不是纯粹的社团，只是纯粹的 hackers，以纵横交叠的各大下属团队部门为核心，以无限的科创脑洞为动力，立足技术与创新的交汇，共同创造团体与个人的未来前景。坚持心中的微光，梦想在这里起航，让自己在浙理成为更好的自己。

下属各部门

  • 网安部

    ZSTUCA网安部致力于逆向分析、web安全、密码学、数字取证、渗透、反编译等方向的研究，以及向全校师生普及相应的知识。团队成员有很多都是参加国内各大CTF比赛的大佬。

  • 软研部

    ZSTUCA软研部致力于Web前后端、桌面应用、手机App等软件的开发，帮助全校有需要的师生编写一些帮助他们解决问题的小软件。计协的官网以及一些相关的服务也都是由他们维护的。同时他们也负责向全校师生进行开发方面的教学与科普。

  • 运维部

    ZSTUCA运维部门致力于操作系统、嵌入式开发等偏向计算机底层知识的学习与探究，以及向全校师生普及。他们负责运营计协所有服务器，保障计协服务器正常运行，并为各开发项目提供持续部署支持(2023届由于服务器是软研部部长自己提供的，所以由软研部负责此项工作)。

  • 数媒部

    ZSTUCA数媒部负责计协各项活动宣传方案的拟定和宣传，提高协会知名度。负责运维协会的各类平台账户以及推送。负责面向全校师生的数字媒体方面软件的教学工作。

  • 硬件部

    协会对外最具人气的队伍。聚集了一群热心的技术帝。精通电脑维修。包括电脑清灰、重装系统、故障修复等。定期进行摆摊义务维修。承接校内外各种电脑维修工作。技术成熟，效率高，已经成为计协的一面标志性的旗帜。多年以来，零零散散维修了近3500台电脑，解决了近师生近4500人次（该数据为2017年的早期数据，目前数据仍在不断上升中）的电脑问题。受到了我校师生的广泛好评。
